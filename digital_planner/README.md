# Digital Planner
A Python-based digital planner with a graphical user interface (GUI) built using Tkinter. The app helps users stay organized by tracking tasks, courses, daily notes, and journal entries, all with persistent storage so your data is saved between sessions.

# Features
- **To-Do List:** add, view, and delete tasks; tasks are saved automatically
- **Courses:** track courses, add or remove them; courses are saved automatically
- **Calendar:** daily notes and tasks; automatically saves notes to a file
- **Journal:** write daily journal entries; saved automatically

# Installation
1. Make sure Python 3.x is installed  
2. Tkinter usually comes with Python  
3. Clone or download this repository: git clone https://github.com/anikapawa/digital-planner.git
4. Navigate into the project folder: cd digital_planner

# Usage
Run the app with Python: python planner.py
- Use the **To-Do tab** to manage tasks  
- Use the **Courses tab** to track courses  
- Use the **Calendar tab** to write daily notes  
- Use the **Journal tab** to write journal entries  
- All data is saved automatically in `data.json` and restored when you reopen the app  

# Technologies Used
- Python 3.x  
- Tkinter for GUI  
- JSON for persistent storage
