# Movie Recommender
A Python-based movie recommendation system using user-based collaborative filtering. The app generates personalized movie suggestions based on users' genre preferences and past ratings.

# Features
- **User-based recommendations:** suggests movies based on similarity between users  
- **Genre preferences:** considers user-selected genres for tailored suggestions  
- **Interactive rating system:** rate movies to create a user preference profile  
- **Recommendation export/import:** save preferences and generate recommendations  
- **Optional visualizations:** analyze user or dataset trends with charts

# Installation
1. Make sure Python 3.x is installed  
2. Clone or download the full portfolio ZIP from GitHub: https://github.com/anikapawa/github_portfolio/archive/refs/heads/main.zip  
3. Navigate into the project folder: cd github_portfolio/movie_recommender` 

# Usage
1. Rate a few movies via search or the "Random Movies" tab.  
2. Click "Export My Preferences" after rating five or more movies to download your preferences file.  
3. Upload the preferences file back into Codio.  
4. Run `python make_recommendations.py` to generate recommendations (saved in the `out` folder).  
5. Load your recommendations in the “Recommender” tab to view them.

# Technologies Used
- Python 3.x
- JSON for user data and movie information
- User-based collaborative filtering algorithm
